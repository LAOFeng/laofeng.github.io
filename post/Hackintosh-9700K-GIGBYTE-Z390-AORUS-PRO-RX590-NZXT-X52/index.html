# Hcakintosh, NZXT

之前其实了解过黑苹果（Hackintosh），但后来买了MBP之后就没再研究。今年突然想配一台比较好的台式在家，恰逢618即将要来，便开始折腾了起来。

装机配置

黑果对硬件有一定要求，一般Intel 的CPU 和 AMD 独显卡比较好黑，但目前三A平台也逐渐有大神在推进。有土豪神装（如果是土豪，为啥不直接白果💻），也可以平民装，可以参考TonyMacx86根据需要配置。
